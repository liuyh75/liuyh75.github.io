# 收益统计软件

这是一个基于浏览器的收益统计软件，可以帮助用户记录和分析个人收益情况。

## 功能特点

- ✅ 收益记录管理：添加、编辑、删除收益记录
- ✅ 日历视图：直观查看每日收益情况
- ✅ 统计分析：按渠道、时间维度统计收益
- ✅ 投资分配：管理不同渠道的投资比例
- ✅ 数据管理：导出、导入、备份数据
- ✅ 用户认证：账号密码注册和登录
- ✅ 设置功能：修改密码、主题设置等

## 如何使用

### 本地使用

1. 直接在浏览器中打开 `index.html` 文件即可使用。
2. 首次使用需要注册账号密码。
3. 可以通过"设置"按钮修改密码和应用设置。

### 上传到 GitHub

1. 在 GitHub 上创建一个新的仓库。
2. 将本目录下的所有文件上传到 GitHub 仓库中。
3. 在 GitHub 仓库的 "Settings" 中，找到 "Pages" 选项。
4. 在 "Source" 下拉菜单中选择 "main" 分支，然后点击 "Save"。
5. 等待几分钟后，GitHub 会生成一个访问链接，通过该链接即可在网络上访问此软件。

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chart.js (用于数据可视化)

## 数据存储

- 用户数据和应用设置存储在浏览器的 localStorage 中。
- 收益记录和投资分配数据也存储在 localStorage 中。
- 可以通过"数据管理"页面导出和导入数据。

## 注意事项

- 由于数据存储在浏览器的 localStorage 中，清除浏览器缓存会导致数据丢失，建议定期备份数据。
- 本软件仅在浏览器端运行，不会将数据上传到任何服务器。
- 密码以明文形式存储在 localStorage 中，仅用于本地使用，不建议在公共电脑上使用。

## 浏览器兼容性

支持所有现代浏览器，包括 Chrome、Firefox、Safari、Edge 等。
